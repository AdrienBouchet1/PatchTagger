## Start with PatchTagger

### Installation 
First, create a conda environment and install required dependencies : 


```bash 
conda env create -f requirements.yaml --name PatchTagger
conda activate PatchTagger
``` 


### Run Patchtagger






```bash 
python PatchTagger.py
```
